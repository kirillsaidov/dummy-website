# A dummy website
A FastAPI application serving a "Coming Soon" page.

## Run

### Local development

```bash
# Install dependencies
python3 -m venv venv
./venv/bin/pip install -r requirements.txt

# Run the server
./venv/bin/python main.py
```

Visit `http://localhost:8081`

### Deployment

```bash
docker compose up
```

Visit `http://localhost:8081`
